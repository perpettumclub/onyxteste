# Feature Roadmap

## V1: Launch (MVP)
**Goal:** Functional platform for a single expert to manage their launch.

- [ ] **Authentication:** Login/Register with Supabase Auth.
- [ ] **Multi-tenancy:** Create/Select "Client Area" (Tenant).
- [ ] **Dashboard:**
    - [ ] Stats widgets (Tasks, Progress).
    - [ ] AI Daily Briefing (Mocked or Real via Edge Function).
- [ ] **Kanban:**
    - [ ] CRUD Tasks.
    - [ ] Drag & Drop status update.
- [ ] **CRM (Leads):**
    - [ ] List View.
    - [ ] Add/Edit Lead.
- [ ] **Member Area:**
    - [ ] View Modules/Lessons.
    - [ ] Video Player (Embed Youtube/Vimeo/Storage).
    - [ ] Mark as Completed.
- [ ] **Finance:**
    - [ ] Set Goals.
    - [ ] View Transactions (Manual Entry).
    - [ ] Split Calculator.

## V2: Growth & Automation
**Goal:** Enhanced automation and team collaboration.

- [ ] **Team Management:** Invite members to Tenant (Editor/Viewer roles).
- [ ] **Real AI Integration:**
    - [ ] Chat with Leads (AI Agent).
    - [ ] Auto-generate Task descriptions.
- [ ] **Billing Integration:** Stripe/Hotmart webhook integration for real transactions.
- [ ] **Advanced Analytics:** Charts for Lead conversion rates.

## V3: Scale
**Goal:** Mobile app and offline capabilities.

- [ ] **Mobile App:** React Native / PWA.
- [ ] **Offline Mode:** Local-first sync (WatermelonDB or similar).
- [ ] **White-label:** Custom domain and branding for Tenants.
